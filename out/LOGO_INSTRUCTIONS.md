# Инструкция по добавлению логотипа Rokets

## Быстрый старт

1. **Скачайте логотип из Google Drive:**
   - Откройте: https://drive.google.com/file/d/1Ak2AEGt3TKIB9-ZKkX0C63IRElJuHDV-/view?usp=sharing
   - Нажмите "Скачать" (Download)

2. **Сохраните файл:**
   - Если файл в формате SVG, переименуйте в `rokets-logo.svg`
   - Если файл в формате PNG, переименуйте в `rokets-logo.png`
   - Поместите файл в папку: `commerce/public/`

3. **Готово!**
   - Логотип будет автоматически использоваться на сайте
   - Никаких изменений в коде не требуется

## Поддерживаемые форматы

- **SVG** (рекомендуется) - `rokets-logo.svg`
- **PNG** - `rokets-logo.png`

## Где используется логотип

- Шапка сайта (Navbar)
- Футер (Footer)
- Open Graph изображения (для социальных сетей)

## Примечание

Если файл логотипа не найден, будет использоваться встроенный логотип-заглушка (ракета).
