# Scripts для работы с листингами

## Экспорт листингов

Экспортирует все листинги в CSV файл для редактирования.

### Установка зависимостей

```bash
npm install -D tsx
```

### Запуск экспорта

```bash
npx tsx scripts/export-listings.ts
```

Это создаст файл `listings-export.csv` в корне проекта со всеми листингами.

### Формат CSV

Файл содержит следующие колонки:
- `handle` - уникальный идентификатор листинга (не изменяйте!)
- `title` - название товара
- `description` - описание товара
- `price` - цена в AED
- `category` - категория (например, "Furniture")
- `subcategory` - подкатегория (например, "sofas-couches")
- `location` - локация продавца (например, "Dubai")
- `mainImage` - URL главного изображения
- `additionalImages` - дополнительные изображения через запятую

### Редактирование

1. Откройте `listings-export.csv` в Excel, Google Sheets или другом редакторе
2. Обновите описания товаров
3. Замените URL изображений на новые
4. Добавьте дополнительные изображения через запятую в колонке `additionalImages`

### Импорт обратно

После редактирования вам нужно будет вручную обновить файл `lib/demo/products.ts` или использовать скрипт импорта (в разработке).

## Примечания

- Не изменяйте колонку `handle` - это уникальный идентификатор
- URL изображений должны быть полными (начинаться с http:// или https://)
- Дополнительные изображения разделяются запятой без пробелов
- После импорта перезапустите dev сервер
