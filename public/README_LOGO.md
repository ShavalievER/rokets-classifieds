# Инструкция по добавлению логотипа Rokets

## Вариант 1: Из Google Drive

1. Откройте ссылку на логотип: https://drive.google.com/file/d/1Ak2AEGt3TKIB9-ZKkX0C63IRElJuHDV-/view?usp=sharing

2. Скачайте файл:
   - Нажмите на кнопку "Скачать" (Download) в правом верхнем углу
   - Или правый клик на файле > "Скачать" / "Download"

3. Сохраните файл:
   - Если файл в формате SVG, переименуйте его в `rokets-logo.svg`
   - Если файл в формате PNG, переименуйте его в `rokets-logo.png`
   - Поместите файл в папку: `commerce/public/`

4. Логотип будет автоматически использоваться! (код автоматически определяет наличие файла)

## Вариант 2: Из Figma

1. Откройте Figma файл: https://www.figma.com/design/evZfExISs74jw5FotuXqY9/%D0%A7%D0%B5%D0%BB%D0%BD%D0%BE%D0%BA-%D0%91%D1%80%D0%B5%D0%BD%D0%B4%D0%B8%D0%BD%D0%B3?node-id=2982-924&t=2Z0LI9PJ90sdHxvD-1

2. Найдите логотип на странице (node-id=2982-924)

3. Выберите логотип (кликните на него)

4. Экспортируйте логотип:
   - Правый клик на выбранном элементе
   - Выберите "Export" или "Экспорт"
   - Выберите формат **SVG** (рекомендуется) или PNG
   - Нажмите "Export" или "Экспорт"

5. Сохраните файл:
   - Назовите файл: `rokets-logo.svg` (если SVG) или `rokets-logo.png` (если PNG)
   - Сохраните файл в папку: `commerce/public/`

6. Логотип будет автоматически использоваться!

## Поддерживаемые форматы

- **SVG** (рекомендуется) - `rokets-logo.svg`
- **PNG** - `rokets-logo.png`

## Примечание

Код автоматически определяет наличие файла логотипа в папке `public/`. Если файл найден, он будет использоваться вместо встроенного SVG. Если файл не найден, будет использоваться встроенный логотип-заглушка.
