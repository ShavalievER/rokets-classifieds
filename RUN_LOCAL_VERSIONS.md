# Как запустить локальные версии на localhost:3000 и localhost:3005

## Быстрый запуск

### Вариант 1: Запустить обе версии одновременно (рекомендуется)

Откройте два терминала в папке `commerce`:

**Терминал 1 - Статичная версия (localhost:3000):**
```bash
cd commerce
npm run build
npm run test:static
```

**Терминал 2 - Динамичная версия (localhost:3005):**
```bash
cd commerce
npm run dev:dynamic
```

Или:
```bash
node server-local.js
```

### Вариант 2: По отдельности

#### Статичная версия (localhost:3000)

1. Соберите статичную версию (если еще не собрали):
```bash
cd commerce
npm run build
```

2. Запустите статичный сервер:
```bash
npm run test:static
```

Или:
```bash
npm run serve:static
```

Или напрямую:
```bash
node test-static-local.js
```

Откроется на: **http://localhost:3000**

#### Динамичная версия (localhost:3005)

Запустите динамичный сервер:
```bash
npm run dev:dynamic
```

Или:
```bash
npm run serve:dynamic
```

Или напрямую:
```bash
node server-local.js
```

Откроется на: **http://localhost:3005**

## Что проверять

### Статичная версия (localhost:3000)
- ✅ Загружается главная страница
- ✅ Навигация работает
- ✅ Страницы товаров открываются
- ✅ Все стили загружаются

### Динамичная версия (localhost:3005)
- ✅ Загружается главная страница
- ✅ API routes работают (`/api/account/addresses` и т.д.)
- ✅ Навигация работает
- ✅ Страницы товаров открываются

## Важные заметки

1. **Статичная версия требует сборки** - перед запуском нужно выполнить `npm run build`
2. **Динамичная версия работает в dev режиме** - изменения кода применяются автоматически
3. **Обе версии можно запускать одновременно** - они используют разные порты
4. **Если порт занят** - измените порт в соответствующем файле:
   - Статичная: `test-static-local.js` (строка 7: `const port = 3000;`)
   - Динамичная: `server-local.js` (строка 18: `const port = 3005;`)

## Скрипты из package.json

- `npm run test:static` - запустить статичную версию
- `npm run serve:static` - то же самое
- `npm run dev:dynamic` - запустить динамичную версию
- `npm run serve:dynamic` - то же самое
- `npm run build` - собрать проект для статичной версии
